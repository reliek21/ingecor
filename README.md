## Ejecutar Local

Clonar el proyecto

```bash
  git clone git@github.com:reliek21/ingecor.git
```

Ir al directorio del proyecto

```bash
  cd ingecor
```

Instalar las dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run start
```
o

```bash
    npm run dev
```
  
## Correr Test

Correr los Test, siguiento el comando

```bash
  npm run test
```
