## Carcteristicas

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform

  
## Tecnologias

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express

  
## Ejecutar Local

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Correr Test

To run tests, run the following command

